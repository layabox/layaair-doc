Core. LoaderManager

LoaderManager Class is used to load resources and data in batches.

Batch loader, single case, can be accessed by Laya.loader.

Multi threading (default 5 threads), 5 priority (0 fastest, 4 slowest, default 1) When a resource fails, the load is retried at the lowest priority (property retryNum decides to retry several times). If the attempt fails, the call to the complete function is called and returns null
